Git and Github commands

As you can see below those are git and github commands so memories them(section1 branch)

  1. git branch # list branches
  2. git branch(name) # create a new branch
  3. git checkout (name) # switch branch
  4. git switch(name) # (newer way) switch branch
  5. git checkout -b(name) # create + switch
  6. git merge(branch) # merge branch into current
  7. git branch -d(name) # delete branch
